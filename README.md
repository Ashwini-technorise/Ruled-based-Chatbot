# Ruled-based-Chatbot

## Description
A simple rule based chatbot that responds to user inputs using predefined conditions, demonstrating basic programming concepts like loops, conditional statements, and functions.

## Features
- Responds to basic inputs like "hello", "how are you", "bye"
- Uses conditional statements (if-else)
- continuous interaction using loops
- Simple and user-friendly

## Technologies used
- Python

## How to run
1. Open terminal
2. Run the program: Python task3.py
